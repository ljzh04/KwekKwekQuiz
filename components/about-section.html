About Kwek Kwek Quiz

Kwek Kwek Quiz is a versatile quiz application designed for learning and fun. Create your own quizzes using JSON, generate them with AI, or load pre-made examples.

Version: 1.0.0 (Legacy Design)

Author: ljzh04

This application utilizes several open-source technologies, including:

  • Tailwind CSS
  • Google Gemini API
  • Marked (for Markdown)
  • Highlight.js (for code syntax highlighting)
  • Katex (for LaTeX rendering)
  • js-yaml (for YAML to JSON)
