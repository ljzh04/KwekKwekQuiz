Welcome to the Kwek Kwek Quiz documentation!

Getting Started

To use the AI Quiz Generation feature, you'll need a Gemini API Key. You can obtain one from Google AI Studio. Paste this key into the API Key field in the "App" section or manage it in "Settings".

Creating Quizzes

You can create quizzes by:

  • Pasting valid JSON into the "Quiz JSON" textarea. Use the "Format JSON" button to prettify it or convert YAML to JSON.
  • Pasting text into the "Quiz JSON" textarea and clicking "Generate JSON" to use AI to create a quiz (requires API Key).
  • Loading a sample quiz using the "Sample" button.

JSON Format Example:

[
  {
    "type": "multiple-choice",
    "question": "What is the capital of France?",
    "options": ["Berlin", "Madrid", "Paris", "Rome"],
    "correct": 2 
  },
  {
    "type": "true-false",
    "question": "The earth is flat.",
    "correct": false
  },
  {
    "type": "fill-in-the-blank",
    "question": "The chemical symbol for water is __.",
    "correct": "H2O"
  }
]

Ensure your JSON follows the structure of an array of question objects. Each object needs `type`, `question`, and `correct`. Multiple-choice questions also need an `options` array (correct is the 0-based index).

Taking Quizzes

Once a quiz is loaded, navigate through questions using "Next" and "Previous". For multiple-choice and true/false, clicking an option submits it. For fill-in-the-blank or identification, type your answer and click "Submit" or press Enter.

Managing Quizzes

Enter a "Quiz Name" and click "Save" to store your current quiz in the browser's local storage. Select a saved quiz from the dropdown to load it. Use "Delete" to remove the selected saved quiz.
